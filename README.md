# Car Price Prediction MLOps Project

## Overview

An end-to-end Machine Learning application that predicts used car prices using FastAPI, Streamlit, MySQL, and Scikit-Learn.

## Features

- Train model using MySQL data
- FastAPI prediction service
- Streamlit frontend
- Automatic prediction logging
- MySQL integration
- Ready for retraining pipeline

## Tech Stack

- Python
- Scikit-Learn
- FastAPI
- Streamlit
- MySQL
- Pandas
- Joblib

## Run

pip install -r requirements.txt

uvicorn app:app --reload

streamlit run streamlit_app.py
